## Definition of context engineering
Is the craft to provide to an AI model the best possible context information it requires to provide the best possible answer.

In an agentic use case there is also a histoty (KV cache) so that the statement becomes:

**_Definition_** Context Engineering is the craft to provide to an AI model the best possible context information it requires to provide the best possible answer by best using the cache of the model instance.
